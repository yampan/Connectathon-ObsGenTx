# Connectathon-ObsGenTx
generate Observation resource and transfer to server
    
## 使い方（uv 環境）
  このツールは、'uv'を使用して、ローカル環境を汚さずに実行します。
  
### 1. 使い捨てで実行する
  uvx --from git+https://github.com/yampan/Connectathon-ObsGenTx.git ObsGenTx
  
### 2A. Localにインストールする場合
  uv tool install git+https://github.com/yampan/Connectathon-ObsGenTx.git
  
### 2B. 実行
  obsgentx
